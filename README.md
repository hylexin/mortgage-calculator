# MortgageCalc

A static mortgage calculator with taxes, insurance, PMI, HOA fees, extra payments, biweekly payments, payoff date, and amortization schedule.

The production-ready static files are in `outputs/`.

## Deploy with GitHub Pages

In GitHub repository settings, enable Pages with:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/outputs`

Before publishing, replace `https://yourdomain.com` in `outputs/robots.txt` and `outputs/sitemap.xml` with the final site URL.
